# FrontRES Core Notes

This folder contains the original FrontRES design and implementation notes.

## Layout

- `contracts/design_contract.md`: method contract and research boundary.
- `plans/engineering_plan.md`: implementation plan.
- `checklists/modification_checklist.md`: active implementation checklist history.
- `paper/`: paper-facing method notes.
- `logs/`: raw discussion or decision history.

Segment Replay specific notes should go to `note/frontres_segment_replay/`.
